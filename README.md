# CIS 228: Hello Expo Project

## Description

A React Native mobile application built using the Expo framework for CIS 228. This project serves as an introductory "Developer Portal" that demonstrates core front-end concepts including layout architecture, state management with hooks (`useState`), user input validation, and native alert handling.

## Features

- Interactive UI using custom React Native components (`View`, `Text`, `TextInput`, `Button`, `Image`).
- Platform-specific keyboard avoidance for a seamless user experience.
- Real-time state management.
- Custom styling using `StyleSheet.create`.

## Installation

To run this project locally, ensure you have Node.js installed, then follow these steps:

1. Clone the repository:

   ```bash
   git clone [https://github.com/YOUR-USERNAME/cis228-hello-expo.git](https://github.com/YOUR-USERNAME/cis228-hello-expo.git)

   ```

2. Navigate into the project directory:
   cd cis228-hello-expo

3. Install the dependencies:
   npm installnpm install
